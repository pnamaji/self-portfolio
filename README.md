## 🎮 Core Interactive Features & Experience

* **Cinematic Startup Sequence:** A high-tech terminal-style preloader simulates live system diagnostic logs (PostgreSQL mounts, Celery worker pool mapping, Docker container isolation), instantly capturing recruiter attention before dissolving into the main layout.
* **True 3D Spatial Depth & Gyroscopic Tilt:** Implements advanced Vanilla JavaScript cursor-coordinate trackers. Moving the mouse leans and skews the active content cards and backdrop arrays in real-time 3D depth layers.
* **Nested Smart Scroll Navigation Engine:** Designed with fluid full-screen snap-slides (`100vh`), ensuring exactly **one major topic per screen**. If a content block contains large datasets (like the 9-projects showcase), scrolling first scrolls the internal text; only when completely exhausted does it smoothly slide to the next macro section.
* **Vibrant Cosmic Cyber Theme:** Replaces dark, monotonous black grids with a premium gradient mesh of Deep Sapphire, Obsidian Blue, Royal Violet, and Neon Cyan accents.
* **Zero-Friction UX:** Completely interactive without annoying password fields or authorization puzzles—optimized for low-latency scrolling and point-and-click operations.
